# 输入日志

> 记录人类所有关键输入，保留决策依据

---

## 格式说明

每条记录包含：
- **时间**：输入时间
- **原始输入**：人类原话
- **类型**：需求 / 决策 / 反馈 / 变更 / 灵光
- **阶段**：关联的流程阶段
- **归纳**：AI 一句话总结
